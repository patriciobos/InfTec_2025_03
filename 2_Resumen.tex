	\large{\textbf{\textcolor{black}{Rutinas de configuración y adquisición para interfaz de audio Behringer.}}}\\
	\vspace{1pc}
	\textbf{Patricio Bos}
\end{center}
\bigskip
\begin{center}
    \textcolor{black}{RESUMEN}	
\end{center}
\begin*
    \indent
    \textit{Se presenta el desarrollo de un módulo en Python para configurar y controlar la interfaz de audio Behringer UMC204HD, utilizada en la estación autónoma EAMMRA. El módulo permite la adquisición programática de señales de hidrófonos y fue validado con pruebas automatizadas mediante \texttt{unittest} y \texttt{pytest}. Se alcanzó una cobertura del 96\% sin fallos reportados, confirmando su robustez para aplicaciones de monitoreo acústico en laboratorio y campo.}
\end*
\bigskip
\begin{center}
    \textcolor{black}{ABSTRACT}
\end{center}
\begin*
    \indent
    \textit{A Python module for configuring and controlling the Behringer UMC204HD audio interface, used in the autonomous EAMMRA station, is presented. The module enables programmatic acquisition of hydrophone signals and was validated through automated testing using \texttt{unittest} and \texttt{pytest}. A 96\% test coverage was achieved with no reported failures, confirming its robustness for acoustic monitoring in both lab and field environments.}
\end*
